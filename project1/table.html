TABLE


Defination:
In HTML, the table tag is used to create a table for displaying data in rows and columns. It is made up of rows tr and cells, which can be headers th or data td. Tables can also include thead, tbody, and tfoot for better structure. Attributes like border, cellspacing, and cellpadding (now mostly done with CSS) control appearance. Tables are commonly used for tabular data, not for page layout in modern web design.

Table tags:

  • Table: Defines the main table structure. It acts as a container for all table elements.
  • tr: Stands for "table row" and defines a row in the table.
  • th: Table header cell. It is bold and centered by default. Used for headings.
  • td: Table data cell. Contains regular content in a row.
  • thead: Groups the header content (tr with th) in a table.
  • tbody:Groups the main body content of the table.
  • tfoot: Groups footer content (like totals) in a table.
  • caption: Adds a title or description above the table.
  • col:Specifies column properties, used inside colgroup.
  • colgroup: Groups one or more columns to apply common styles or settings.

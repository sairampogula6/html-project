MULTI MEDIA

Defination:

Multimedia refers to content that combines multiple forms of media like text, audio, images, video, and animations. It is used to make websites, applications, and presentations more interactive and engaging. In web development, multimedia enhances user experience by conveying information in different formats. It supports learning, entertainment, communication, and marketing effectively. Modern browsers provide built-in support for multimedia elements. Examples include music players, online videos, photo galleries, and embedded web apps.

TAGS:

  • audio - Used to embed sound/music in a webpage..
  • video - Used to display video content with controls like play/pause.
  • img - Displays images such as JPG, PNG, or GIF on a webpage.
  • iframe - Embeds another HTML page or external content (like YouTube, maps) inside a webpage

Common Attributes Used:

  • src → Specifies the file URL (audio, video, image, or page).
  • controls → Adds play/pause/volume controls (for audio & video).
  • width / height → Sets size of video, image, or iframe.
  • alt → Provides alternate text for images (important for SEO & accessibility).
  • autoplay → Automatically plays audio/video when page loads.
  • loop → Repeats audio or video continuously.

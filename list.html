LIST TAGS

Definition:
The li tag in HTML stands for "list item". It is used to define individual items within a list. It must be placed inside a parent list element like ul (unordered list), ol (ordered list), or menu. Each li element creates one item in that list.


TYPES:

  1. ordered list (ol):

    Defination:
    A list where items are arranged in a specific sequence and are usually numbered.

    Example:

    1. Wake up
    2. Brush teeth
    3. Eat breakfast

  2. unordered list (ul):

    Defination:
    A list where the items are shown with bullet points and the order doesn’t matter.

    Example:

    • HTML
    • CSS
    • JavaScript

  3. nested list :

    Defination:
    A list that is placed inside another list, showing a main item with sub-items underneath it.

    Example:

    1. Fruits
      • Apple
      • Banana
      • mango
    2. Vegetables
      • tamato
      • potato
      • bottle guard

  4. Defination list(dl,dt,dd):

    Defination:
    A list that pairs words or terms with their meanings or explanations, like in a dictionary.

    Example:

    Festival :
    A festival is a special event or celebration held by a group of people, community, or culture, often once a year, to honor traditions, religion, seasons, or important events.

NOTE:

  • Unordered list default bullet is a solid disc (●); when nested, bullets change to a circle (○) then a square (■) for deeper levels.

  • Ordered list default numbering is decimal (1, 2, 3, …); can also use letters (A, B, C) or Roman numerals (I, II, III).
